# Twitter Clone One

A fully responsive frontend [Twitter Clone](twitter-clone-one.ntabucejo.vercel.app/).
Coded by [Nikko Abucejo](https://github.com/ntabucejo).

## Strategies 

* CSS Flexbox
* CSS Grid
* CSS (Fixed / Sticky / Absolute) Positions
* Mobile First Approach
